A. 구조체 정보 추가

모든 구조체에 주소, 생일, 메모를 추가하였습니다.
이에 따라서 출력의 모든 정보를 수정하였습니다.


B. 새로운 기능(여러 개의 번호 입력)

- 다양한 전화번호를 추가적으로 입력할 수 있게 만들었습니다.
(EX. 집전화, 직장전화, 휴대폰, 팩스 등등)

- 때문에 구조체 안에 구조체를 집어넣었고, 이를 insert쪽에서 수정하기 위해
삼중 포인터를 사용하여 main에서 insert 메소드로 삼중 포인터를 넘겼습니다.

- 이로인해 insert 쪽 코드가 실행되며, 추가적으로 번호를 입력할 경우,
삼중 포인터를 이용해서 NumberInfo 구조체를 하나씩 더 realloc해주었습니다.


C. 파일을 이용한 DB 구현

- txt파일에서 초기 정보를 읽어오고, 종료시 txt파일에 정보를 저장하도록 하였습니다.
